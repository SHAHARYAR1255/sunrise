# Privacy Policy — sunrise

_Last updated: 2026-06-27_

**sunrise does not collect, store, transmit, or share any personal data.**

The plugin is a single slash command (`/sunrise:set`) that runs locally inside your own Claude Code session. When you run it, it:

1. Reads the start time you type and your system timezone — locally — to compute a schedule.
2. Creates a scheduled "routine" on **your own** Anthropic/Claude account, using **your own** credentials, via Claude Code's built-in routine feature.

That is the only action it performs. Specifically, sunrise:

- Sends **no** data to the plugin author or any third party.
- Contains **no** analytics, telemetry, tracking, or network calls to author-controlled servers.
- Stores **no** data itself.

The routine it creates runs on Anthropic's cloud infrastructure under your account and is therefore governed by Anthropic's Privacy Policy: https://www.anthropic.com/legal/privacy

Questions or concerns: open an issue at https://github.com/shaharyar1255/sunrise/issues
